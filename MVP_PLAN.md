# DoctorRx — MVP Plan (Patient Compliance Tracker)

## Product Overview

**One-line pitch:** A beautifully designed, low-friction system that transforms patient behavior into actionable insights for doctors.

**Target conditions:** Diabetes & Obesity (chronic, requires daily tracking)

**Two users:**
- **Doctor** — adds patients, selects condition (auto-assigns default template), toggles which questions to show per patient, views compliance dashboards
- **Patient** — taps a magic link, answers the questions their doctor enabled (dynamic, template-driven)

## Compliance Engine — Detailed Logic

### Input
- All check-ins for a `doctor_patient_id` in a given date range

### Score Calculation

```typescript
function calculateCompliance(
  checkIns: CheckIn[],
  enabledQuestions: string[],
  templateQuestions: TemplateQuestion[],
  days: number = 7
) {
  // Only yes_no questions count toward compliance score
  const yesNoKeys = templateQuestions
    .filter(q => q.type === "yes_no" && enabledQuestions.includes(q.key))
    .map(q => q.key);

  const metrics: Record<string, { done: number; total: number; label: string }> = {};
  for (const key of yesNoKeys) {
    const q = templateQuestions.find(tq => tq.key === key)!;
    metrics[key] = { done: 0, total: 0, label: q.label };
  }

  for (const checkIn of lastNDays(checkIns, days)) {
    for (const key of yesNoKeys) {
      metrics[key].total++;
      if (checkIn.responses[key] === true) metrics[key].done++;
    }
  }

  // Overall = total yes answers / total possible across all yes_no questions
  const totalDone = Object.values(metrics).reduce((sum, m) => sum + m.done, 0);
  const totalPossible = Object.values(metrics).reduce((sum, m) => sum + m.total, 0);
  const overall = totalPossible > 0 ? Math.round((totalDone / totalPossible) * 100) : 0;

  return { overall, metrics };
}
```

### Trend Detection

```typescript
function detectTrend(
  checkIns: CheckIn[],
  enabledQuestions: string[],
  templateQuestions: TemplateQuestion[]
) {
  const current = calculateCompliance(checkIns, enabledQuestions, templateQuestions, 7);
  const previous = calculateCompliance(checkIns, enabledQuestions, templateQuestions, 14);

  const diff = current.overall - previous.overall;
  const THRESHOLD = 10; // percentage points

  if (diff > THRESHOLD) return "improving";
  if (diff < -THRESHOLD) return "worsening";
  return "stable";
}
```

### Insight Generation (Rule-Based, Dynamic)

```typescript
function generateInsights(
  checkIns: CheckIn[],
  enabledQuestions: string[],
  templateQuestions: TemplateQuestion[]
): string[] {
  const insights: string[] = [];
  const last7 = lastNDays(checkIns, 7);

  // No check-in streak
  const missedDays = 7 - last7.length;
  if (missedDays >= 3) {
    insights.push(`No check-in for ${missedDays} days`);
  }

  // Per yes_no question: check for missed days
  const yesNoQuestions = templateQuestions
    .filter(q => q.type === "yes_no" && enabledQuestions.includes(q.key));

  for (const q of yesNoQuestions) {
    const missed = last7.filter(c => c.responses[q.key] !== true).length;
    if (missed >= 3) {
      insights.push(`Missed "${q.label}" ${missed} times this week`);
    }
  }

  // Weight trend (if weight is an enabled number question)
  if (enabledQuestions.includes("weight")) {
    const weights = last7.filter(c => c.responses.weight != null).map(c => c.responses.weight as number);
    if (weights.length >= 2 && weights[weights.length - 1] > weights[0]) {
      insights.push("Weight trending up");
    }
  }

  return insights;
}
```

## Notification System (Phase 3)

### Feature Flag
```env
WHATSAPP_ENABLED=true
```

### Flow
1. Vercel Cron triggers `/api/reminders/send` daily
2. Fetch all doctor_patients with reminders enabled
3. Filter by reminder_time (within current hour window)
4. Send WhatsApp via Twilio to each patient's phone
5. Update `last_sent_at`

### Message Template
```
Hi Ravi 👋
Time for your daily check-in with Dr. Sharma

👉 [Check-in now]: https://doctorrx.in/p/[token]
```

### Cron Schedule
```json
// vercel.json
{
  "crons": [{
    "path": "/api/reminders/send",
    "schedule": "0 * * * *"
  }]
}
```

Runs every hour. The API filters patients whose `reminder_time` falls within the current hour.
